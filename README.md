# Prerequisite
  Install pflags from https://github.com/chahal-p/pflags
# Installation
  1. Download a version from https://github.com/chahal-p/pbash_utils/releases
  1. Unzip the file
  1. Navigate to extracted directory
     ```sh
     cd pbash_utils
     chmod +x install.sh
     ```
  1. ```sh
     ./install.sh
     ```
  1. For bash, add below to .bashrc
     ```
     source "$HOME/.local/bin/pbu_init.bash"
     ```
     For zsh, add below to .zshrc
     ```
     source "$HOME/.local/bin/pbu_init.zsh"
     ```
